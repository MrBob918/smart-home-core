# Smart Home Core

Микросервисная архитектура для голосового управления умным домом. Управляется через голосовые запросы с телефона с удобным интерфейсом.

## 📁 Структура проекта

```
smart-home-core/
├──stt-server (сервер перевода speach to text)
├──http-server (сервер для хранения запросов из мобильного приложения, передачи на расшифровку в stt-server, а после в llm-server)
├──llm-server (обрабатывает запросы в json, и решает выполнение каких либо дейстивий)
├──proto (в хранятся .proto файлы для контейнеров)
└──docker-compose.yml(для аркестарции серверами)
(возможно немного изменений в будущем но основной скелет будет таким)

---
smart-home-core/
└─stt-server
  ├──src/
  │  ├──stt/ реализация stt библиотеки с ease-to-use vosk
  │  └──stt-main.cpp main flow файл
  ├──include/
  │  ├──stt/ зоголовок stt библиотеки с ease-to-use vosk
  │  └──vosk-include/ файлы vosk (загруажется автоматически)
  ├──Dockerfile (TODO)
  └──proto (компилируется автоматически)

---
smart-home-core/
└──http-server
   └──TODO (архетекрура может менятся в процессе разработки)
   
---
smart-home-core/
└─llm-server
  └─TODO (архетекрура может менятся в процессе разработки)
```
